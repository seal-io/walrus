Walrus是一个开源的应用管理平台，可以简化在任意基础设施上应用服务的部署和管理。它帮助平台工程师为开发人员构建黄金路径，并赋予开发人员自服务能力。

# 快速入门

```shell
sudo docker run -d --privileged --restart=unless-stopped -p 80:80 -p 443:443 sealio/walrus
```

在浏览器中打开 `https://<服务器IP或域名>`
